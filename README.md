# Stallman Interjection Text Generator<br>

---

## Building:<br>

GNU/Linux:<br>
```sh
gcc interjection.c -o interjection
```

Windows:<br>
```bat
echo Stallman does not approve.
```

macOS: <br>
```sh
echo Stallman does not approve.
```

---

## Running:<br>

GNU/Linux:<br>
```sh
./interjection
```

Windows:<br>
```bat
echo Stallman does not approve.
```

macOS: <br>
```sh
echo Stallman does not approve.
```

---

## Examples:

```sh
./interjection
```
```sh
./interjection --linux "$(uname -s)"
```
```sh
./interjection --gnu Ganoo --linux Loonix
```
